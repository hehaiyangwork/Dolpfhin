## bluedolphin
A Node Command Line Tool to auto generate and build a web structure .

## Install

```bash
// local
cd bluedolphin
npm link

// npm
npm install bluedolphin -g
```

## Usage
```bash
dolphin [init [project name] | build | watch] [options]
```

Then
```bash
cd <project name>
npm install
dolphin build
```

Now you can develop your app...

## Options

